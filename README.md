## 📘 Descrição
Este projeto aplica técnicas de **Clusterização (K-Means)** para segmentar clientes com base em características comportamentais e demográficas.  
O objetivo é identificar grupos distintos de clientes e gerar insights que possam apoiar estratégias de marketing, retenção e personalização.

---

## 🎯 Objetivo
Agrupar clientes em segmentos com perfis semelhantes, permitindo análises direcionadas e tomadas de decisão mais assertivas.

---

## 📊 Dataset
- Variáveis: idade, renda, score de gasto, etc.  
- Fonte: dataset público amplamente utilizado em estudos de clusterização.  
- Tamanho: aproximadamente 200 registros.  
- Tipo: dados numéricos e categóricos.

---

## 🧪 Metodologia
1. Padronização dos dados  
2. Aplicação do **Elbow Method** para determinar o número ideal de clusters  
3. Treinamento do modelo **K-Means**  
4. Visualização dos clusters em 2D  
5. Interpretação dos grupos e insights de negócio

---

## 📈 Resultados
- Identificação de **3 clusters principais** com perfis distintos de clientes  
- Visualização clara dos grupos  
- Insights úteis para estratégias de segmentação e fidelização
